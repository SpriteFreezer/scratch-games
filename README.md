# scratch-games
Делаю поделки используя скретч.

Любую игру можно запустить на сайте : 
https://scratch.mit.edu/projects/editor/

0) Скачать файл игры 

1) Перейти по ссылке :
https://scratch.mit.edu/projects/editor/

2) Выбрать игру :
Файл -> Загрузить с компьютера

3) Справа сверху развернуть на весь экран

4) Играть =)
